# Rock_Paper_Scissors

Classis Rock, Paper, Scissor game created for the web browser.

About Game:
- You play against the computer
- The first one to 5 points wins

Rules:
- Paper beats Rock
- Rock beats Scissors
- Scissors beats Paper

Tech used:
- Version control: Git/Github
- IDE: Visual Studio Code
- Website: 
    * HTML
    * CSS
    * JavaScript
